# OpenClaw — COMMANDS.md

| Comando WhatsApp | Acción | Toca PJN | Requiere GPT |
|---|---:|---:|---:|
| `Expediente: listar` | Lista expedientes locales | No | No |
| `Expediente: estado pjn 12345/2024` | Estado local | No | No |
| `Expediente: capturar pjn 12345/2024` | Ejecuta script externo e ingiere | Sí | No |
| `Expediente: novedades pjn 12345/2024` | Captura y resume estado | Sí | No |
| `Expediente: informe pjn 12345/2024` | Captura, normaliza, cura, analiza, valida y reporta | Sí | Sí |

Los comandos son deliberadamente pocos. La exuberancia de interfaces es la forma en que los bugs aprenden a reproducirse.
